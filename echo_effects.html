Echo implications Mnemonic Name Parameter(s) Description

Standard FT2 effects


Done 0xy Arpegio 1st halftone, 2nd halftone First tick the note is played, the next tick — note + x halftones, the next tick — note + y halftones, then repeat from the beguinning. For instance: C-1 037: first is played C1, then C1 + 3 ht = Eb1, then — C1 + 7 ht = G1 and then again C1 and so on...
Done 1xx/2xx* Portamento up/down Speed This command is used to slide the sample pitch up or down. This is done using the speed value. If Amiga frequence table is used, the sliding will be non-linear (the speed depends on the frequency).
Done 3xx* Tone portanemto Speed This command is used together with a note, and will slide to its frequency. If glissando mode is set (by E3x command), the frequency will be rounded to the nearest halftone
Done 4xy* Vibrato Rate, depth Adds (frequency) vibrato to the channel with a specified rate and speed. Set vibrato mode command (E4x) can be used to change the vibrato waveform (see below).
Possible 5xy* Tone portamento + volume slide Up speed, down speed (see volume slide (Axy)) This command execute both tone portamento and volume slide. The speed is used for the volume slide.
Possible 6xy* Vibrato + volume slide Up speed, down speed (see volume slide (Axy)) This command execute both tone portamento and volume slide. The speed is used for the volume slide.
Possible 7xy* Tremolo Rate, depth The same like vibrato (4xy) but affects the sound amplitude rather then the frequency.
00, 80, FF Done 8xy Set panning position Position 0x00 — leftmost, 0xff — rightmost.
NOT POSSIBLE 9xx Sample offset Offset This command should be used together with a note. The sample will be played from (offset * 0x100) instead of zero.
Done Axy* Volume slide Up speed, down speed Slides the current volume up or down. Either up speed or down speed should be zero
NOT POSSIBLE Bxx Position jump Position This command will cause jump to the selected song postion and play pattern from the beginning
Done Cxx Set volume Volume Sets the current volume of the track where the command is, just like volume column. The volume should not be greater than 0x40
Only D00 Done! Dxx Pattern break Pattern position This command will cause jump to the next pattern and play from the selected position
NOT POSSIBLE E0x Set filter mode (Amiga only) Mode I don't have Amiga so I can say nothing about how this command works.
Possible E1x/E2x* Fine portamento up/down Speed This command works as portamento up/down, but the sliding occurs only one time per line.
Unnecessary E3x Set glissando mode Mode If mode = 1, the resulting frequency after implementing this command will be rounded to the nearest halftone.
Posible, but unlikely E4x Set vibrato mode Mode Mode: 0 — sine, 1 — ramp down, 2 — square. If you add 4 to the mode, the waveform will not be retrigged when a new instrument is played
NOT POSSIBLE E5x Set fine-tune Tune This command should be used together with a note. It will cause another fine-tune value to be used.
Possible but unlikely E6x Pattern loop Count If count is zero, the beginning of the loop will be specified. When a non-zero value is used, the pattern will be looped from a loop start by a given number of times.
Possible but unlikely E7x Set tremolo mode Mode See E4x (set vibrato control), but for the tremolo waveform.
Possible and TODO! E9x Retrig note Interval Retrigs note with the specified interval.
Possible, but unlikely EAx/EBx* Fine volume slide up/down Speed This command works as usual volume slide up/down, but the sliding occurs only one time per line.
Possible and TODO! ECx Note cut Tick Cut the note at the specified tick. Note that it will only set the volume to zero, and the sample will still be played.
Possible and TODO! EDx Note delay Ticks This command will delay the note by the selected number of ticks.
Possible, but unlikely EEx Pattern delay Lines This command will delay pattern by the selected number of tracker lines
Done Fxx Set speed Speed or BPM This command will set the speed or BPM value of the song. If value is less than 0x20, the speed will be changed. Othervise, the BPM value will be changed.
Difficult and unlikely Gxx Set global volume Volume Sets the global volume. The value should not exceed 0x40.
Difficult and unlikely Hxy* Global volume slide Up speed, down speed Works exactly as volume slide, but it slides the global volume instead.
NOT APPLICABLE Lxx Set envelope position Position Changes the envelope position (measured in ticks, as in the instrument editor).
NOT POSSIBLE Pxy Panning slide Right speed, left speed Slides the panning position. Works like the volume slide.
